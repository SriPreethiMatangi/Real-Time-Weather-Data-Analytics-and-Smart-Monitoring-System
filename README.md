Project Overview:
The Real-Time Weather Data Analytics and Smart Monitoring System is a Python-based web application 
that collects live weather information using the OpenWeatherMap API, stores weather records, performs
analytics, and presents the results through a simple dashboard.
The system helps users monitor current weather conditions and analyze weather trends in real time.

Features:
-Fetches real time weather data using OpenWeatherMap API
-Displays: Temperature, Humidity, Atmospheric Pressure, Wind Speed, Rainfall
-Weather analytics and monitoring
-Stores weather information in a database
-Simple web interface using Flask
